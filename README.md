# DT193G Moment 3
Frontend application for managing music album collection.  
  
Written in TypeScript with [Vue.js](https://vuejs.org/) and interacts with a [FastAPI backend application](https://github.com/AstronomyOverdrive/dt193g_moment2).  
Components are used for improved maintainablitiy and the abilitiy to reuse components if needed.  
Uses Vue.js router for SPA functionality and props/emits to pass data between components.  
Fetch API is used to communicate with the backend.  
[Tailwind](https://tailwindcss.com/) is used for styling.
